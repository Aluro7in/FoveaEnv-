## What is FoveaEnv?

**FoveaEnv** is a reinforcement learning environment for training **privacy-aware attention navigation agents**.

Inspired by the constraints of on-device AI like Meta's Ray-Ban Surfer, FoveaEnv challenges agents to navigate efficiently while respecting privacy zones — a benchmark designed for the **real world**, not just the lab.
